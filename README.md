# Material + Jekyll
## Getting started
1. `git clone https://github.com/nikhilnayak98/blog.git`
2. `cd blog`
3. Configure the `_config.yml` file as needed
4. `jekyll serve`

## Options
Customize your options within the `_config.yml` file.

+ Theme
  - Green
  - Blue
  - Orange
  - Purple
  - Grey
+ Fixed Navigation
  - True
  - False
