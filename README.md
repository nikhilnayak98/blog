## Getting started
1. `git clone https://github.com/nikhilnayak98/personalwebsite-v2.git`
2. `cd personalwebsite-v2`
3. Configure `_config.yml`
4. `jekyll serve`

## Features

+ Theme
  - Green
  - Blue
  - Orange
  - Purple
  - Grey
+ Fixed Navigation
  - True
  - False
+ Progressive Web App
+ Comments (Disqus Comment Hosting Service)
